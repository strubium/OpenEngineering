# OpenEngineering

OpenEngineering is a Cleanroom remake of Immersive Engineering to allow for continued maintenance and compatibility on 1.12.2

## What?
While Immersive Engineering is a well-loved mod, the 1.12.2 version has lingering bugs and limitations that are unlikely to be addressed.

## How does OpenEngineering handle IE's Code?
The source code for Immersive Engineering are licensed under a proprietary license so OpenEngineering is using Cleanroom design, a method of copying a design by reverse engineering and then recreating it without infringing. 

OpenEngineering cannot look at Immersive Engineering's source code, nor use it for the remake. 


## How does OpenEngineering handle IE's Assets?
The assets for Immersive Engineering are also licensed under a proprietary license (Blu's License of Common Sense) so OpenEngineering cannot distribute them.

OpenEngineering does however have a tool for downloading textures from [the ImmersiveEngineering GitHub](https://github.com/BluSunrize/ImmersiveEngineering/tree/1.13pre) but this not required for any functionality.
